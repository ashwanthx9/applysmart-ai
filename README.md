# 💼 ApplySmart AI: CV & Job Match Assistant

## 📌 Overview

ApplySmart AI is an interactive web application that helps job seekers compare their CV with a target job description and get role-specific resume improvements.

Users can upload their CV, paste a job description, and receive a structured report showing:

- CV match score
- Matching skills
- Missing or weak skills
- Resume profile improvements
- Skills section rewrite
- Experience bullet point improvements
- Project section improvements
- Final resume focus areas

The tool is designed to give short, practical, copy-paste friendly CV suggestions instead of generic advice.

## ▶️ How to Use

1. Open the app link.
2. Select an analysis mode from the sidebar:
   - Gemini AI
   - OpenRouter AI
   - OpenAI GPT
   - Claude AI
   - Basic Analysis
3. Paste the required API key if using an AI mode.
4. Upload your CV / resume as a PDF.
5. Paste the full job description.
6. Click **Analyse Resume**.
7. Review the generated CV improvement report.

## 🧠 AI Providers

The app supports multiple analysis options:

- Google Gemini
- OpenRouter
- OpenAI GPT
- Anthropic Claude
- Basic Analysis without API

Each user can paste their own API key in the sidebar. API keys are not stored in the app.
